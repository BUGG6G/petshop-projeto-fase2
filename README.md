# PetShop AmigoFiel - Fase 2

## Sobre o Projeto

O PetShop AmigoFiel é um sistema web desenvolvido para a disciplina de Fundamentos de Sistemas Web. O objetivo do projeto é simular um pet shop online, oferecendo catálogo de produtos, serviços de banho e tosa, cadastro de clientes e agendamento de serviços.

## Funcionalidades

* Catálogo de produtos para pets
* Carrinho de compras ilustrativo
* Formulário de cadastro de clientes e pets
* Página de agendamentos
* Agendamento de banho e tosa através de modal
* Validação de datas e horários para agendamento
* Navegação entre páginas
* Interface responsiva utilizando Bootstrap

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap 5

## Estrutura do Projeto

* `pagina_principal.html` - Página inicial do sistema
* `formulario.html` - Cadastro de clientes e pets
* `agenda.html` - Visualização de agendamentos
* `carrinho.html` - Carrinho de compras
* `estilo.css` - Estilos do sistema
* `logica.js` - Funcionalidades JavaScript
* `imagens/` - Imagens utilizadas no projeto

## Recursos de Acessibilidade

O sistema utiliza o atributo `alt` nas imagens para fornecer descrições textuais, auxiliando usuários que utilizam leitores de tela.
